# Ayam Penyet Best - Landing Page

A modern, responsive landing page for Ayam Penyet Best restaurant located at Mayang Mall, Terengganu.

## Features

- **Responsive Design**: Mobile-first approach with Bootstrap 5
- **Smooth Navigation**: Fixed navbar with scroll effects
- **Interactive Menu Carousel**: Browse through 5 main dishes
- **Customer Testimonials**: Feedback carousel with auto-rotation
- **Online Booking**: Two-column reservation form
- **Modern UI**: Clean design with custom color palette

## Sections

1. **Hero Section**: Full-screen introduction with restaurant branding
2. **Menu Section**: Oval transition with interactive dish carousel
3. **Feedback Section**: Customer testimonials with auto-rotating carousel
4. **Booking Section**: Online reservation form
5. **Footer**: Contact information and social media links

## How to Use

1. Open `index.html` in a web browser
2. Navigate through sections using the navbar or scroll
3. Browse menu items using arrow buttons
4. Fill out the booking form to make a reservation
5. View customer feedback in the testimonials section

## Interactive Features

- **Navbar**: Transforms on scroll with background color change
- **Menu Carousel**: Navigate between 5 dishes with visual transitions
- **Feedback Carousel**: Auto-rotates every 5 seconds with touch support
- **Smooth Scrolling**: All navigation links scroll smoothly to sections
- **Form Validation**: Client-side validation for booking form

## Responsive Breakpoints

- **Mobile**: < 576px (Single column layout)
- **Tablet**: 576px - 991px (Adjusted spacing and layouts)
- **Desktop**: > 992px (Full multi-column layouts)
